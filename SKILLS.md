# 七鲜商品素材采集与AI视频生成

完整的七鲜商品从素材采集到AI视频生成的工作流。

## 工作流程概览

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│  1. 采集主图     │ -> │  2. 生成分镜     │ -> │  3. 即梦生成视频 │
└─────────────────┘    └─────────────────┘    └─────────────────┘
         │                      │                      │
         v                      v                      v
  下载5张主图素材      生成1900字内脚本       每10分钟监控状态
  保存至指定目录       舌尖中国风格           生成后自动下载
```

---

## 第一步：采集商品主图

### 目标
从 7fresh.m.jd.com 下载指定商品的所有主图。

### 操作步骤

1. **打开七鲜移动端网站**
   ```
   https://7fresh.m.jd.com
   ```

2. **搜索目标商品**
   - 使用搜索功能找到商品
   - 或直接进入商品详情页 URL

3. **进入商详页**
   - 确认商品名称和 SKU ID
   - SKU ID 通常在 URL 中（如 `skuId=308443`）

4. **下载主图**
   - 使用浏览器截图或图片下载功能
   - 下载所有商品主图（通常5张）

### 文件保存规范

**目录结构：**
```
/Users/clawdbot/7freshTools/
└── 商品名-skuId/
    ├── 商品名-skuId-1.jpg
    ├── 商品名-skuId-2.jpg
    ├── 商品名-skuId-3.jpg
    ├── 商品名-skuId-4.jpg
    ├── 商品名-skuId-5.jpg
    └── 商品名-skuId-分镜脚本
```

**命名规则：**
- 文件夹：`商品名-skuId`（如：`春菜红香椿100g-308443`）
- 主图：`商品名-skuId-序号.jpg`
- 分镜脚本：`商品名-skuId-分镜脚本`
- 视频素材：`商品名-skuId-视频素材.mp4`

### 质量检查

下载完成后执行检查：
```bash
# 检查图片数量和大小
ls -lh ~/7freshTools/商品名-skuId/

# 验证图片格式
file ~/7freshTools/商品名-skuId/*.jpg
```

**检查要点：**
- [ ] 主图数量正确（通常5张）
- [ ] 图片格式为 JPG
- [ ] 图片大小合理（>100KB）
- [ ] 图片可以正常打开

## 第二步：生成视频分镜脚本

### 目标
根据商品特点生成15秒视频分镜脚本，用于即梦AI视频生成。

### 风格指南

**食品类商品（舌尖上中国风格）：**
- 风格：东方时令美学、匠心传承
- 配音：中年/青年女声，温润醇厚
- 语调：讲述感，慢语速（120字/分钟）
- BGM：古筝+笛子+轻鼓点

**其他品类：**
- 根据商品特性自由发挥
- 核心是勾起购买欲望
- 突出核心卖点和使用场景

### 分镜结构

15秒视频分为5个镜头：

| 镜头 | 时长 | 内容 |
|------|------|------|
| 第1镜 | 0-3秒 | 商品引入，产地/原料特写 |
| 第2镜 | 3-6秒 | 核心工艺/制作过程 |
| 第3镜 | 6-10秒 | 产品展示，使用场景 |
| 第4镜 | 10-13秒 | 品尝/使用体验 |
| 第5镜 | 13-15秒 | 品牌露出，CTA |

### 分镜脚本格式

每个镜头需包含：

```
第N镜 | 起始-结束秒
================================================================================
画面内容：
[详细描述画面内容]

运镜手法：
[运镜方式、光线、特效]

卖点文字（叠加）：
【主标】xxx
字体：xxx
字号：xxx
颜色：xxx
位置：xxx

【副标】xxx
字体：xxx
...

旁白配音（情绪标注）：
「旁白文案」
语速指导、情绪标注

参考素材：
图X、图X
```

### 脚本要求

- 字数控制在 **1900字以内**
- 每个镜头都要有「主标」文字叠加
- 配音要有情绪标注和语速指导
- 参考素材要对应下载的主图序号

### 质量检查

生成完成后检查：
- [ ] 总时长15秒（各镜头时长相加）
- [ ] 字数在1900字内
- [ ] 包含5个镜头
- [ ] 每个镜头有主标文字
- [ ] 有配音和情绪标注
- [ ] 风格统一

**详细示例参考：** [references/storyboard-example.md](references/storyboard-example.md)

## 第三步：即梦AI视频生成

### 目标
在即梦平台上传图片和分镜脚本，生成15秒商品视频。

### 操作步骤

1. **打开即梦网站**
   ```
   https://jimeng.jianying.com/ai-tool/generate
   ```

2. **选择创作类型**
   - 创作类型：视频生成
   - 模型：Seedance 2.0 Fast
   - 生成方式：全能参考（不是首尾帧）
   - 比例：9:16
   - 时长：15s

3. **上传参考图片**
   - 从下载的5张主图中选取 **5张最具代表性** 的
   - 上传作为全能参考

4. **粘贴分镜脚本**
   - 将生成的分镜脚本完整粘贴到提示词区域
   - 确保脚本内容完整

5. **开始生成**
   - 点击生成按钮
   - **记录当前商品的 SKU ID**（用于后续监控）

### 视频生成监控

**监控频率：** 每10分钟检查一次

**监控方式：**
```bash
# 打开即梦生成页面
open https://jimeng.jianying.com/ai-tool/generate

# 检查视频状态
# - 如显示"生成中" → 继续等待
# - 如显示"完成"且有播放按钮 → 立即下载
```

**判断生成成功：**
- 视频预览可以正常播放
- 页面显示"完成"或"下载"按钮

## 第四步：即梦视频下载

### 方式一：页面点击下载（推荐）

**操作步骤：**

1. **进入视频详情页**
   - 在即梦生成列表找到目标视频
   - 点击进入播放界面

2. **点击下载按钮**
   - 鼠标悬停在视频上
   - 点击右上角显示的「⬇ 下载」按钮
   - 视频自动下载到浏览器默认目录

3. **移动并重命名**
   ```bash
   mv ~/Downloads/下载的视频.mp4 \
      ~/7freshTools/商品名-skuId/商品名-skuId-视频素材.mp4
   ```

### 方式二：获取视频URL直接下载

**适用场景：** 批量下载、自动化脚本

**操作步骤：**

1. **进入视频播放页**
   - 同方式一的步骤1

2. **获取视频直链**
   ```javascript
   // 在浏览器控制台执行
   document.querySelector('video').src
   ```

3. **使用curl下载**
   ```bash
   curl -L -o "商品名-skuId-视频素材.mp4" "视频URL"
   ```

### 文件命名

```
商品名-skuId-视频素材.mp4

示例：
- 春菜红香椿100g-308443-视频素材.mp4
- 陈皮红豆沙-308446-视频素材.mp4
```

## 注意事项

### 主图下载
- 七鲜移动端网站需要用手机模拟器或真实手机访问
- 确保下载的是高清原图
- 主图通常包含：商品全貌、细节特写、使用场景、包装展示等

### 分镜脚本
- 字数必须控制在1900字内（即梦限制）
- 食品类优先使用「舌尖上中国」风格
- 每个镜头都要有明确的视觉描述

### 即梦生成
- 上传的5张图片要有代表性，覆盖视频需要的场景
- 生成时间根据排队情况，通常几分钟到几十分钟
- 视频URL有有效期，生成后尽快下载

### 视频下载
- 即梦视频默认下载到浏览器下载目录
- 下载后及时移动到项目目录并重命名
- 视频格式为MP4，可直接用于剪辑
